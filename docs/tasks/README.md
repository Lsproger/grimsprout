# Tasks

Задачи проекта GrimSprout. Каждый файл — одна задача.

## Формат имени файла
`{phase}-{NNN}-{slug}.md` — например `phase3-001-ollama-client.md`

## Статусы
- `backlog` — запланировано
- `in-progress` — в работе
- `done` — завершено
- `blocked` — заблокировано (указать чем)

Сводка по фазам — в [../plan.md](../plan.md).
