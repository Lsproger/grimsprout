# Фирменная «загробная» стилистика ответов

**Фаза**: 4
**Статус**: backlog
**Приоритет**: low
**Зависимости**: нет

## Описание

Придать всем текстовым ответам бота единый «загробный» стиль — мрачно-иронический, в духе склепа и некроботаники. Затрагивает системный промпт для LLM и статические строки хендлеров.

## Критерии готовности

- [ ] Определён и задокументирован tone-of-voice (3–5 примеров фраз для разных контекстов)
- [ ] Системный промпт (`config/prompts/system_undertaker.md`) обновлён с учётом стилистики
- [ ] Статические ответы в хендлерах (полив, удобрение, пересадка, ошибки) переписаны в новом стиле
- [ ] Сообщения FSM (`/new`, `/edit`) переработаны
- [ ] Стиль применён последовательно: нет «нейтральных» фраз рядом с «загробными»

## Заметки

- Tone-of-voice: некробот-садовник, ведущий Журнал Теней. Мрачный, но без агрессии. Примеры:
  - «Полив зафиксирован в анналах склепа.»
  - «Растение опознано. Что угодно усопшему?»
  - «Коммит совершён. Тьма запомнила.»
- Не затрагивать admin-команды и сообщения об ошибках безопасности — оставить нейтральными
- Связать с `config/prompts/changelog_style.md` для единого стиля changelog-записей
