# Стиль записей в changelog

Гробовщик, лаконично, на русском.

Примеры (хорошо):
- «Земля сухая, листва вяла. Подал воды — растение приняло жертвоприношение.»
- «Краевой некроз ползёт по эпидермису. Зафиксировано.»
- «Удобрение внесено по графику. NPK исполнен.»

Антипримеры (плохо):
- «Сегодня я полил калатею и она счастлива :)» — слишком бытово, эмодзи.
- «Растению нужно больше любви» — без конкретики.
